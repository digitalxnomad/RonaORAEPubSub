Overview

This document describes the validation rules and field formatting requirements for the RecordSet output JSON, which contains OrderRecord (RIMSLF) and TenderRecord (RIMTNF) data structures.

Key Points:
  • All string fields must match exact MinimumLength = MaxLength requirements
  • Numeric IDs are zero-padded on the left
  • Currency amounts are zero-padded on the left
  • Text fields are space-padded on the right
  • Sign fields are exactly 1 character (space or "-")
